Bill Payment service

How to run:
1. On window:
	run file: runme.bat

2. On linux
	run file: runme.sh

Syntax
1. CASH_IN [number] : Add amount into balance
2. LIST_BILL: list all bill
3. PAY [billId]: Pay for 1 bill
4. PAY [billId 1] [billId 2] [billId 3]: Pay for multi bill
5. DUE_DATE: list bill not pair
6. SCHEDULE [billId] [date]: set schedule pay for bill
7. LIST_PAYMENT: list all bill has payment
8. SEARCH_BILL_BY_PROVIDER [Provider]: list bill by provider
